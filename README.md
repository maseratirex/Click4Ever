# Click4Ever
An upcoming powerful auto clicker creator, allowing the user to create multiple customizable auto clickers. Development currently stalled due to a debilitating bug caused by macOS's security measures: macOS has <a href="https://stackoverflow.com/questions/53103394/robot-mousemove-does-not-work-at-all-in-mac-os-x">trouble</a> enabling Java's AWT Robot class to modify mouse input and output.
